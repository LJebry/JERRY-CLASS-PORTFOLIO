# Jerry Class Portfolio

A single-page portfolio site built for CISC 3310 – Multimedia Programming. It highlights my background, featured projects, experience, and houses links to class assignments.

## Features
- Hero section with direct contact links (email, phone, LinkedIn, GitHub)
- Projects grid with concise descriptions and tech stacks
- Assignments section for each course deliverable
- Experience and education snapshots for quick review

## Tech Stack
- HTML5 for semantic structure
- CSS3 for layout, responsive grid, and typography
- Vanilla JavaScript (`script.js`) reserved for interactive enhancements

## Getting Started
1. Clone the repo and open `index.html` in any modern browser.
2. Adjust content inside `index.html` to reflect new work or assignments.
3. Update styling in `style.css` if you need additional themes or layouts.

## Repository Structure
```
├── index.html      # Page markup
├── style.css       # Visual styles and layout rules
├── script.js       # Placeholder for interactive behavior
└── .vscode/        # Optional workspace settings
```

## Assignments
Each assignment lives inside the “CISC 3310 – Multimedia Programming Assignments” section within `index.html`. Replace the placeholder text/link for each card with the actual work once it is ready.

## Contributing / Tracking Issues
This is a personal class project, but feel free to open issues or submit pull requests with improvements. For security scanning, VS Code is configured to auto-select the Snyk organization defined in `.vscode/settings.json`.
