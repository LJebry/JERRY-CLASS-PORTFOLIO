# Ocean Echo Explorer: Quick Instructions

1. Open the project page at `assignment4/index.html`.
2. Click **Open Web App** to launch `assignment4/app.html`.
3. Choose a topic from the dropdown menu.
4. Use:
   - **Play Topic Audio** for narration,
   - **Read Description** for full text-to-speech,
   - **Toggle Quick Fact** for extra content.
5. Click **Install App** when the button appears to install the PWA.

For local testing, run a local server:

```bash
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000/assignment4/index.html`
- `http://localhost:8000/assignment4/app.html`
