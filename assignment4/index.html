Ocean Echo Explorer

A multimedia learning app about marine life, built with HTML5, CSS, JavaScript, JSON, a manifest, and a service worker.

Open Web App Back to Portfolio

What This App Does

  • Dynamically loads topic data from data/topics.json.
  • Builds the topic menu in JavaScript from the JSON dataset.
  • Displays image, description, and quick fact for each topic.
  • Plays narration with Web Speech API.
  • Supports install prompt and offline caching with service worker.

App Screenshots

How to Install

  1. Open the app from the web version link.
  2. Wait for the Install App button to appear.
  3. Click Install App and accept the browser prompt.
  4. Launch it from your device home screen or app launcher.

If the install button does not appear, the browser may not support PWA install for the current context.
